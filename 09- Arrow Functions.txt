En javaScript existe otra manera de escribir funciones

para ello podriamos asignar la funcion a una variable

ejemplo 

        const esMayorDeEdad = function(persona){
            ... //do Something
        }

este tipo de estructura se le conoce como funcion anonima        


otra forma de escribirlo, es con una Arrow function

        const esMayorDeEdad = (persona)=>{  <-- nos ahorramos la palabra function y tan solo utilizamos una flecha
            ... //do Something
        }
